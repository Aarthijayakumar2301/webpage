HTML Tag Reference Table

Tag Description output
<img> Embeds an image Image Tag
<a> Defines a hyperlink. Anchor tag
<p> Defines a top-level heading Paragraph tag
<h1> Defines a top-level heading Heading tag
<ul> Defines an unordered (bulleted) list Unordered List tag
<ol> Defines an ordered (numbered) list Ordered List tag
<table> Defines a table Table tag
<form> Defines an input form Form tag
<input> Defines an input field Input tag
<button> Defines a clickable button Button tag
<video> Embeds a video file Video tag
<audio> Embeds an audio file Audio tag
<br> Inserts a single line break BR tag
<hr> Creates a horizontal line HR tag
<iframe> Embeds another webpage Iframe tag
